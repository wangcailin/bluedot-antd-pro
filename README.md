# Bluedot for Ant Design Pro

基于[Ant Design Pro](https://pro.ant.design)脚手架二次封装。

## 开始使用

#### 1. 创建项目

```bash
pnpm create bluedot your-project-name
```

#### 2. 进入安装

```bash
pnpm i
```

#### 3. 开发调试

```bash
pnpm start:dev
```

## 目录结构规范

遵循 [@ant-design](https://pro.ant.design/zh-CN/docs/folder) 最佳实践
